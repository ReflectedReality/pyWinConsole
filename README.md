## Simple WinApi wrapper for comfortable working with console. 

### Methods:
```python
  Console.configure_console()
  Console.set_title()
  Console.get_window_size()
  Console.clear()
  Console.hide_cursor()
  Console.show_cursor()
  Console.set_cursor_position()
  Console.write()
  Console.write_line()
  Console.set_text_color()
  Console.reset_colors()

[See an example.py file](https://github.com/ReflectedReality/pyWinConsole/blob/main/example.py)
